- **Regla de Acrónimos**: Al mencionar una sigla por primera vez, escribe primero el significado completo y luego el acrónimo entre paréntesis (ej. "Row Level Security (RLS)", "Single Page Application (SPA)", "Progressive Web Application (PWA)", "Data Definition Language (DDL)").
